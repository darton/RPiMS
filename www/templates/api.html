Start Setup

To download all current data in JSON format, use the following address:

http://{{ request.host.split(':')[0] }}/api/data/?all=show

http://{{ request.host.split(':')[0] }}/api/data/all

In order to obtain data only for a selected sensor or a group of sensors, the address should be filled in with a parameter, e.g .:

http://{{request.environ['SERVER_ADDR']}}/api/data/?sensors=show

CPU sensor

http://{{request.environ['SERVER_ADDR']}}/api/data/?cpu=show

GPIO sensors

http://{{request.environ['SERVER_ADDR']}}/api/data/?gpio=show

BME280 sensors

http://{{request.environ['SERVER_ADDR']}}/api/data/?bme280=show

Show data from CPU and BME280 sensors

http://{{request.environ['SERVER_ADDR']}}/api/data/?cpu=show&bme280=show
