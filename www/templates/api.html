Home Setup

To download current data in JSON format, use the following address:

http://{{ request.host.split(':')[0] }}/api/data/all

To order to obtain data only for a selected sensor or only from sensors, the address should be filled in with a parameter, e.g .:

All sensors

http://{{ request.host.split(':')[0] }}/api/data/sensors/all

CPU sensor

http://{{ request.host.split(':')[0] }}/api/data/sensors/cpu

BME280 sensors

http://{{ request.host.split(':')[0] }}/api/data/sensors/bme280

DS18B20 sensor

http://{{ request.host.split(':')[0] }}/api/data/sensors/ds18b20

Door sensors

http://{{ request.host.split(':')[0] }}/api/data/sensors/door

Motion sensors

http://{{ request.host.split(':')[0] }}/api/data/sensors/motion

Weather Station

http://{{ request.host.split(':')[0] }}/api/data/sensors/weather-station

System

http://{{ request.host.split(':')[0] }}/api/data/system

Config

http://{{ request.host.split(':')[0] }}/api/data/config
