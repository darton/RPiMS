Start Setup

To download all current data in JSON format, use the following address:

http://{{ request.host.split(':')[0] }}/api/data/all

In order to obtain data only for a selected sensor or a group of sensors, the address should be filled in with a parameter, e.g .:

All sensors

http://{{request.environ['SERVER_ADDR']}}/api/data/sensors/all

CPU sensor

http://{{request.environ['SERVER_ADDR']}}/api/data/sensors/cpu

Door sensors

http://{{request.environ['SERVER_ADDR']}}/api/data/sensors/door

BME280 sensors

http://{{request.environ['SERVER_ADDR']}}/api/data/sensors/bme280

DS18B20

http://{{request.environ['SERVER_ADDR']}}/api/data/sensors/ds18b20

Motion sensors

http://{{request.environ['SERVER_ADDR']}}/api/data/sensors/motion

Settings

http://{{request.environ['SERVER_ADDR']}}/api/data/settings

System

http://{{request.environ['SERVER_ADDR']}}/api/data/system
